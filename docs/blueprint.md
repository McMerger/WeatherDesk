# **App Name**: WeatherDesk

## Core Features:

- City Weather Fetch: Fetch current weather data from OpenWeatherMap API based on user-provided city name.
- Kotlin JavaFX Firebase: A desktop weather app in Kotlin using JavaFX UI and Firebase backend.
- Fetch Weather by City: Fetch weather from OpenWeatherMap API by city input
- Forecast Display: Show current and multi-day forecast with icons and clean layout
- Responsive UI: Responsive UI and loading/error feedback
- Store Last Location: Store last-used location in Firebase
- Multi-Day Forecast: Display a multi-day weather forecast, including daily high/low temperatures and weather condition icons.
- Last Used Location: Store the last searched city location in Firestore for quick access.
- UI Display: Show current conditions (temperature, humidity, wind speed) with appropriate weather icons. Layout will be simple and easy to read.
- Responsive Design: Ensure the user interface is responsive and adapts well to different screen sizes. Users can also rate how correct a suggested weather forecast is and store that information in the backend.
- Loading & Error Feedback: Display loading indicators while fetching data and provide informative error messages in case of API failures.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to evoke a sense of clear skies and calmness.
- Background color: Light gray (#F0F8FF) to provide a clean and unobtrusive backdrop.
- Accent color: Orange (#FFA500) to highlight important information like temperature or warnings.
- Body and headline font: 'PT Sans', a humanist sans-serif font, to maintain legibility and a touch of personality, suitable for all UI elements.
- Use clear, weather-related icons sourced from a reliable icon set (e.g., Font Awesome or Weather Icons) to visually represent weather conditions.
- Utilize a clean and modern layout, with a focus on readability and intuitive information hierarchy.
- Incorporate subtle animations to provide feedback (e.g., loading spinners) and enhance the user experience (e.g., smooth transitions).
